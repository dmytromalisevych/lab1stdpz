# Voting System (Система голосування)

Проєкт розроблено в рамках лабораторної роботи з дисципліни "Конструювання програмного забезпечення" (КОП). Це веб-застосунок для створення опитувань та голосування, написаний на Python (FastAPI) з використанням Jinja2 шаблонів.

## ⚙️ Конфігурації та вимоги
- **Backend:** Python 3.10+, FastAPI, SQLAlchemy, SQLite (база даних створюється автоматично).
- **Frontend UI / Storybook:** Node.js, npm, Vite, Bootstrap 5.
- **Документація:** pdoc.

## 🚀 Базові команди для запуску

**1. Запуск серверної частини (FastAPI):**
\`\`\`bash
source venv/bin/activate
python src/main.py
\`\`\`

**2. Генерація документації (pdoc):**
Щоб згенерувати HTML-документацію для Python-коду, виконайте:
\`\`\`bash
source venv/bin/activate
python -m pdoc src/main.py -o docs
\`\`\`

**3. Запуск Storybook (UI Компоненти):**
Щоб запустити середовище тестування компонентів:
\`\`\`bash
npm run storybook
\`\`\`
