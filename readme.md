# Voting System (Система голосування)

Проєкт розроблено в рамках лабораторної роботи з дисципліни "Конструювання програмного забезпечення" (КОП). Це повноцінний локальний веб-застосунок для створення опитувань та проведення голосування, написаний на Python (FastAPI) з використанням Jinja2 шаблонів для рендерингу на сервері.

## 🚀 Встановлення та налаштування

Для розгортання проєкту на вашому локальному комп'ютері виконайте наступні кроки в терміналі.

### 1. Отримання коду
Склонуйте репозиторій на свій комп'ютер та перейдіть у папку проєкту:
```bash
git clone [https://github.com/ВАШ_НІК/voting-system.git](https://github.com/ВАШ_НІК/voting-system.git)
cd voting-system
```

### 2. Налаштування бекенду (Python)
Проєкт використовує віртуальне середовище для ізоляції залежностей. Створіть його та встановіть пакети:
```bash
# Створення віртуального середовища
python3 -m venv venv

# Активація (для macOS/Linux)
source venv/bin/activate
# (Для Windows використовуйте: venv\Scripts\activate)

# Встановлення необхідних бібліотек
pip install -r requirements.txt
```

### 3. Налаштування фронтенду (Node.js)
Для роботи UI-компонентів Storybook необхідно встановити npm-пакети:
```bash
npm install
```

## 🛠 Команди для запуску та розробки

**Увага:** Усі наведені нижче команди слід виконувати з кореневої папки проєкту з **активованим віртуальним середовищем**.

### ▶️ 1. Запуск сервера (FastAPI)
Для запуску основного веб-застосунку:
```bash
cd src
python main.py
```
🔗 **Результат:** Сайт буде доступний у браузері за адресою: [http://127.0.0.1:8000](http://127.0.0.1:8000)

### 📚 2. Генерація документації (pdoc)
Для автоматичної генерації або оновлення HTML-документації на основі docstrings:
```bash
cd src
python3 -m pdoc main.py -o ../docs
```
🔗 **Результат:** Згенеровані HTML-файли з'являться у папці `docs/`. Для перегляду просто відкрийте файл `docs/index.html` у вашому браузері.

### 🎨 3. Запуск Storybook (UI Компоненти)
Для запуску ізольованого середовища розробки та тестування фронтенд-компонентів:
```bash
npm run storybook
```
🔗 **Результат:** Інтерфейс Storybook автоматично відкриється за адресою: [http://localhost:6006](http://localhost:6006)
